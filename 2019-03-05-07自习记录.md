## D.07

1.	c++中内存分为5个区，分别是堆、栈、自由存储区、全局/静态存储区和常量存储区；
2.	运算符重载：
赋值运算符，下标运算符，函数调用运算符，成员运算符必须作为成员函数；
流插入，流提取，类型转换运算符，作为友元函数；
单目运算符，复合运算符，作为成员函数；
双目运算符作为友元函数；
3.	并发操作带来的数据不一致性：
丢失修改（两个都改）
不可重复读（前读后改）
读“脏”数据（前改后读）
4.	索引：
建立索引，create unique index stusno on student(sno)
修改索引，alter
删除索引，drop
5.	授权：
Grant select on table student to u1;
Revoke select on table student from u1;
6.	关系数据库系统的查询处理步骤：
查询分析，查询检查，查询优化，查询执行；
7.	检查点：
步骤（找最后一个检查点记录；找正在执行的事务清单；正向扫描日志文件；分两个队列做不同处理）
8.	封锁协议：
一级（防止丢失修改）二级（防止读脏数据）三级（防止不可重复读）；
9.	早睡早起加油。
